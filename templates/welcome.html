Встановлення (мінімум для роботи)

  1. Встановіть Python 3.8+ з python.org і у вікні інсталятора увімкніть Add python.exe to PATH.
  2. Windows: подвійний клік на start_medchar.bat — створиться venv і встановляться залежності з requirements.txt (лише те, що потрібно для цієї програми).
  3. Вручну: python -m venv venv → активуйте venv → pip install -r requirements.txt → python app.py.
  4. Відкрийте в браузері цю сторінку або одразу форму медичної характеристики.

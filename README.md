# МедХар — медична характеристика (Excel only)

Веб-застосунок на Flask для генерації **медичної характеристики** з Excel. Без баз даних.

## Швидкий старт

1. Встановіть [Python 3.8+](https://www.python.org/downloads/) з **Add python.exe to PATH**.
2. **Windows:** подвійний клік **`start_medchar.bat`** — створиться `venv`, встановляться залежності з **`requirements.txt`** (тільки мінімум для `app.py`).
3. Покладіть файли Excel у папку **`data/`** (вона створюється автоматично при першому запуску програми, поруч із `app.py`).
4. У браузері відкриється **http://127.0.0.1:5000/** — вітальна сторінка з інструкцією; звідти можна перейти до форми генерації.

Вручну: `pip install -r requirements.txt` → `python app.py`.

## Файли в `data/`

| Файл | Призначення |
|------|-------------|
| `treatments_YYYY.xlsx` | Назва строго: `treatments_` + 4 цифри року + `.xlsx` (наприклад `treatments_2026.xlsx`). |


Потрібен **хоча б один** річний файл. Колонки: **Прізвище, Ім'я, По батькові** або **ПІБ**.

Файли `*.xlsx` у `data/` у git не потрапляють (`.gitignore`).

## Залежності

Єдиний список — **`requirements.txt`** (Flask, pandas, openpyxl, docxtpl, pymorphy3 тощо). Додаткових наборів пакетів у проєкті немає.
